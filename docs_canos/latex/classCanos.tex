 A posição x de cada cano é baseada na posição do primeiro multiplicado pela index que ele representa no vetor de objetos \doxylink{classCanos}{Canos} vezes o espaço definido de um cano para o outro. A altura é calculada randomicamente entre 50 e 250px. A altura do cano de baixo é calculada a partir do tamanho da tela menos a altura do cano de cima menos a altura do buraco (no qual o passáro passa) menos a altura do "{}chão"{}.
