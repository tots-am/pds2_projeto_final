\doxysubsection{Detailed Description}
Representa o arquivo de texto usado para salvar as informações dos jogadores no fomato key value. 

A classe é responsável por realizar as operações de CRUD no arquivo de texto, e interagir com a classe \doxylink{classScoreBoard}{Score\+Board} Cada linha no arquivo representa um player
